# The Long Way to London · Summer 2026

A self-contained, single-file travel companion app for a ~6-week solo trip from **Taiwan → London** (Jul–Aug 2026): a day-by-day itinerary, a bucket-list reel, a booking tracker, an editable budget, and a "never-lonely" social layer.

**Just open `index.html`** in any browser — no build step, no dependencies. It works offline (photos fall back to gradients), and your checkboxes/budget edits save locally.
